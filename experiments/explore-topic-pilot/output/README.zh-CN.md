# 具身智能三向对照归档

discovery-03-embodied-intelligence 使用同一自然语言 prompt、同一模型和同一 reasoning effort，比对三种研究方式：

```text
output/development/discovery-03-embodied-intelligence/
  default/run-01/
  decision-research-v0.2/run-01/
  explore-topic/run-01/
```

每个正式运行包含：

- research-report.md：模型原始报告；
- run-metadata.md：运行条件与耗时。

三种条件分别代表：

- default：模型默认研究；
- decision-research-v0.2：强调证据纪律、反证与可验证性的厚 Skill；
- explore-topic：强调动态问题地图、延迟收敛和打破首个框架的薄 Skill。

该题参与过 Skill 设计讨论，因此属于开发数据，不是留出验证。
